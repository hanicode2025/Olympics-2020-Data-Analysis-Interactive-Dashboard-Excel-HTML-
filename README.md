# 🏅 Olympics 2020 Data Analysis & Dashboard

## 📌 Overview
This project presents an end-to-end data analysis of the Tokyo 2020 Olympics dataset. It includes data cleaning, processing, and visualization to uncover insights into medal distribution and country performance.

## 🛠️ Tools Used
- Microsoft Excel  
- Data Cleaning & Processing  
- Data Visualization  
- HTML (for dashboard export)

## 📂 Project Structure
- cleaned_data/ → Processed dataset used for analysis  
- dashboard/ → Interactive HTML dashboard  
- reports/ → PDF containing all visualizations  

## 📊 Key Insights
- USA leads in total medals, while China shows strong gold medal efficiency  
- Top 5 countries dominate overall medal distribution  
- Smaller countries exhibit higher efficiency in medal conversion  
- Clear performance gaps between top-tier and mid-tier nations
